将复杂文档转变为结构化数据，只需TextIn xParse智能文档解析。让任意文档的信息都能高效准确流入您的数据库，将非结构化内容转化为可查询、可分析的宝贵数据资产，兼容关系型数据库与向量数据库。

TextIn xParse智能文档解析可以从 PDF 和图像等非结构化文档中提取结构化数据。[文档解析](/xparse/parse-quickstart)可以识别文本、图像、表格和表单字段等元素，并转换成 markdown 和 JSON 格式返回，同时包含精确的页面和坐标参考。[文档抽取](/xparse/extract-quickstart)可以从文档中提取特定的数据字段，支持根据prompt和自定义字段模式抽取。

解析或抽取后的数据可用于下游应用程序，例如知识库、RAG、Agent或其他自定义工作流程。
